The Virtual Brain Python 教程

发表于2026-03-31|更新于2026-04-08
|总字数:1.3k|阅读时长:4分钟|浏览量:

本教程仅介绍 Python 中的 tvb 包（源码：https://github.com/the-virtual-brain/tvb-root），不涉及 The Virtual Brain 作为独立仿真软件的使用。
官方网站：https://www.thevirtualbrain.org/tvb/zwei/
官方文档：https://docs.thevirtualbrain.org。
本文将帮助你在 Python 中使用 The Virtual Brain (TVB) 进行神经动力学建模，并生成仿真颅内脑电（iEEG/sEEG）信号。

TVB 仿真的核心逻辑与组件映射
TVB 的底层逻辑通过常微分方程（ODEs）模拟大尺度脑网络中神经元群体的平均活动。要生成可用的仿真数据，需配置四个核心组件：

  1. 结构连接 (Connectivity)

    • 输入节点间权重矩阵（Weights）和纤维束长度矩阵（Tract Lengths）。
    • 权重决定相互作用强度；结合传导速度可引入时间延迟（Time Delays）。

  2. 神经群体模型 (Neural Mass Model, NMM)

    • 选用适合脑电震荡的模型，建议 Jansen-Rit (JR) 或 Generic2dOscillator。
    • 典型模型：
      • Wilson-Cowan (WC)：经典 E-I 模型，结构简洁、易调参。
      • Epileptor：5 维非线性，适合癫痫发作与棘波模拟。
      • Jansen-Rit (JR)：大脑皮层柱兴奋与抑制交互，适合 sEEG/iEEG，天然产生 Alpha/Beta 震荡。
    • 公式层面将线性迭代转为非线性微分：
      \dot{X}(t)=F(X(t),A_{true}X(t-\tau))+w(t)

  3. 耦合函数 (Coupling Function)

    • 决定一个节点如何经结构连接影响另一个节点。
    • 线性耦合有助于保持拓扑到动态的直接映射（适合 VGAE 特征学习）。

  4. 随机积分器 (Stochastic Integrator)

    • 引入生理噪声，避免仿真信号过于平滑。
    • 推荐 HeunStochastic，通过噪声方差控制信噪比（SNR）。

第二步“构建动力学系统”

真实的颅内脑电绝不是单一纯正弦波（比如纯粹的 Gamma），而是复杂动力学系统的叠加表现：

  • 周期性成分（Periodic component）：多个频段神经震荡（Theta/Alpha/Beta/Gamma），来自局部神经元群体的同步放电。
  • 非周期性成分（Aperiodic component）：宽频噪声（常见 $1/f$ 幂律功率谱），反映大脑整体兴奋/抑制平衡和基础代谢活动。

在 TVB 中复现这种“复合感”，关键在于“非线性群体模型 + 随机噪声驱动”。推荐组合：Jansen-Rit（JR）模型 + 随机积分器（Stochastic Integrator）。

当处于临界点（bifurcation）附近的 JR 模型持续注入高斯白噪声时：

  • 保留固有主导共振（Alpha/Beta）
  • 非线性机制将噪声转换为宽频复合信号（$1/f$ 背景 + 谐波）
  • 网络间耦合使波动在区域之间传播与叠加
